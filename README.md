# Week 10: Text analysis: Sentiment analysis with Twitter Data
This week, we will learn how to perform a sentiment analysis using Twitter data. Sentiment analysis, also referred to as opinion mining, is a method for identifying the emotional tone behind a body of text. 

- Topics covered: 
    - What is sentiment analysis?
    - What are practical applications of sentiment analysis?
    - How to perform sentiment analysis with Twitter data?
    - How to use this method with other text data sources?
